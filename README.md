# 充电桩实时状态查询与使用统计看板

一个功能完整的新能源汽车充电桩管理与数据可视化前端应用，提供充电桩分布查询、状态监控、数据分析和个人充电记录管理等功能。

## 项目介绍

本应用基于 Vue 3 + TypeScript + ECharts + Naive UI 技术栈开发，旨在为新能源汽车用户和充电设施运营管理者提供一站式充电信息服务平台。应用包含充电桩分布地图、充电桩列表查询、统计分析看板、个人充电记录四大核心模块，全面支持响应式布局，适配手机、平板及桌面设备。

## 功能说明

### 1. 充电桩分布地图

- 交互式地图展示（基于 ECharts 散点图模拟）
- 状态标记点颜色区分：🟢 空闲 / 🔴 占用 / ⚫ 离线
- 点击标记点弹出信息卡片，展示：
  - 站点名称与详细地址
  - 实时电价（元/度）
  - 当前空闲枪数 / 总枪数
  - 正在充电枪的实时功率数据（kW）
  - 运营商信息与距离

### 2. 充电桩列表

- 数据表格展示所有充电桩详细信息
- 多条件筛选功能：
  - 按状态筛选（空闲 / 占用 / 离线）
  - 按运营商名称筛选
  - 按最大距离筛选
- 多字段排序功能：
  - 按距离升序 / 降序
  - 按电价升序 / 降序
  - 按使用率升序 / 降序
- 一键重置筛选与排序条件
- 分页展示

### 3. 统计分析看板

- **今日充电量趋势折线图**：展示当日 0-23 时充电量变化趋势，带渐变面积填充
- **各时段使用率热力图**：按（星期 × 小时）二维矩阵展示充电桩使用频率分布，颜色越深表示使用率越高
- **站点充电次数排行榜**：横向柱状图展示各站点充电次数，按降序排列
- 顶部数据概览卡片：今日总充电量、运营站点数、平均使用率、累计订单数

### 4. 个人充电记录

- **历史充电订单列表**：
  - 开始时间、站点名称、充电时长
  - 充电量（kWh）、电费、服务费、总费用
  - 订单状态（充电中 / 已完成）
- **月度充电量统计柱状图**：展示近 6 个月充电量变化
- **费用构成分析饼图**：展示电费与服务费占比
- 顶部数据概览：累计充电量、累计充电费用、充电次数、平均单价

## 响应式设计

应用采用桌面优先（Desktop First）的响应式设计策略，断点如下：

| 断点 | 设备类型 | 布局说明 |
|------|----------|----------|
| > 1024px | 桌面端 | 侧边栏完整展开（240px），4 列统计卡，2 列图表网格 |
| 768px - 1024px | 平板端 | 侧边栏可折叠为图标模式，2 列统计卡，单列图表 |
| < 768px | 移动端 | 侧边栏变为底部 Tab 导航，统计卡 2 列，图表单列，表格横向滚动 |

## 开发注意事项

### 代码规范

1. 所有 Vue 组件必须使用 `<script setup lang="ts">` 语法
2. 组件文件使用 PascalCase 命名（如 `ChargingMap.vue`）
3. 每个组件尽量控制在 300 行以内，超过需拆分为子组件
4. Props、Emits 必须定义完整的 TypeScript 类型
5. 组合式函数（Composables）以 `use` 前缀命名，存放在 `composables/` 目录

### 样式规范

1. 全局样式变量定义在 `src/assets/styles/global.css` 的 `:root` 中
2. 主色：绿色 `#18a058`（新能源环保主题），状态色：空闲绿 / 占用红 / 离线灰
3. 间距统一使用 4/8/12/16/20/24px 等 4 的倍数
4. 响应式通过 CSS 媒体查询实现，断点为 1024px 和 768px

### 数据管理

1. 全局状态统一通过 Pinia Store 管理，不使用 Props 深层传递
2. Mock 数据与业务逻辑完全分离，存放在 `src/mock/` 目录
3. 后续接入真实 API 时，只需替换 Store 中的数据源即可
4. 所有数据模型在 `src/types/index.ts` 中统一定义

### 图表使用

1. 所有 ECharts 图表通过 `BaseChart.vue` 组件统一封装
2. 图表配置项通过 `option` prop 传入，组件内部自动处理初始化、响应式、销毁
3. 无需在业务组件中直接操作 ECharts 实例

## Mock 数据说明

项目使用本地 Mock 数据模拟真实场景，包含：

- **15 个充电桩站点**：分布在北京市各区域，覆盖 5 家主流运营商
- **充电枪数据**：每个站点 4-11 把充电枪，含实时功率数据
- **20 条充电订单**：覆盖近 30 天的充电记录
- **24 小时充电量数据**：模拟真实早晚高峰曲线
- **7 × 24 使用率热力图数据**：工作日与周末差异
- **8 个站点充电次数排行数据**
- **6 个月个人充电统计数据**

如需修改 Mock 数据，请编辑 `src/mock/data.ts` 文件。
