## 1. 产品概述

本产品是一个充电桩实时状态查询与使用统计前端应用，旨在为新能源汽车用户提供便捷的充电桩查找、状态查询、充电记录管理服务，并通过数据可视化帮助用户及运营管理者了解充电设施使用情况。

- 核心价值：解决用户找桩难、信息不透明、充电数据分散等痛点，提供一站式充电信息服务
- 目标用户：新能源汽车车主、充电设施运营管理者

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册（默认） | 查看充电桩分布、筛选排序、查看统计看板、查看个人充电记录 |

### 2.2 功能模块

1. **充电桩分布地图页**：交互式地图、状态标记点、信息弹窗卡片
2. **充电桩列表页**：数据表格、多条件筛选、多字段排序
3. **统计分析看板页**：今日充电量趋势图、时段使用率热力图、站点充电次数排行榜
4. **个人充电记录页**：历史订单列表、个人充电量统计图表、费用统计分析

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 充电桩分布地图 | 地图展示区 | 使用ECharts散点图模拟地图，展示充电桩地理位置分布 |
| 充电桩分布地图 | 状态标记点 | 绿色=空闲、红色=占用、灰色=离线，支持点击交互 |
| 充电桩分布地图 | 信息弹窗卡片 | 点击标记点弹出，展示地址、实时电价、空闲枪数、实时充电功率 |
| 充电桩列表 | 筛选区 | 按状态（空闲/占用/离线）、运营商名称、距离筛选 |
| 充电桩列表 | 数据表格 | 展示充电桩详细信息，支持按距离/电价/使用率升序/降序排序 |
| 统计分析看板 | 今日充电量趋势 | 折线图展示当日0-23时充电量变化趋势 |
| 统计分析看板 | 时段使用率热力图 | 按星期×时段二维展示充电桩使用频率分布 |
| 统计分析看板 | 站点充电次数排行 | 柱状图展示各站点充电次数，降序排列 |
| 个人充电记录 | 历史订单列表 | 展示时间、地点、充电量、费用等订单信息 |
| 个人充电记录 | 充电量统计 | 折线/柱状图展示累计及近期充电量变化 |
| 个人充电记录 | 费用统计 | 饼图展示费用构成，展示累计费用 |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：科技蓝（#18a058 绿色作为主色，配合Naive UI默认主题），体现环保新能源理念
- **辅助色**：状态色（绿色#18a058-空闲、红色#d03050-占用、灰色#5c6268-离线）
- **按钮风格**：圆角矩形，柔和阴影，hover时有轻微上浮效果
- **字体**：系统默认无衬线字体，标题加粗，正文清晰可读
- **布局风格**：侧边导航栏 + 主内容区，卡片式容器承载各模块
- **图标风格**：使用Naive UI内置图标，简约线性风格

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|----------|
| 充电桩分布地图 | 地图区域 | 全宽卡片容器，ECharts图表居中展示，顶部状态图例 |
| 充电桩分布地图 | 信息弹窗 | Naive UI Popover卡片，包含地址、电价、功率等信息标签 |
| 充电桩列表 | 筛选区域 | 行内排列的选择器、输入框，紧凑布局 |
| 充电桩列表 | 数据表格 | Naive UI DataTable，斑马纹、固定表头、可排序列 |
| 统计分析看板 | 整体布局 | 2×2网格布局，每个图表独立卡片，标题+图表 |
| 个人充电记录 | 订单列表 | 表格展示，支持分页 |
| 个人充电记录 | 统计区域 | 左右分栏，左侧充电量趋势，右侧费用构成饼图 |

### 4.3 响应式设计

- **桌面端（>1024px）**：侧边栏展开（240px），主内容区多列网格布局
- **平板端（768px-1024px）**：侧边栏可折叠为图标模式，网格布局调整为2列
- **移动端（<768px）**：侧边栏变为底部Tab导航，图表和表格单列垂直排列，表格支持横向滚动

## 5. 非功能性需求

- **性能**：首屏加载 < 3s，图表渲染流畅无卡顿
- **兼容性**：支持Chrome、Firefox、Safari、Edge最新版本
- **可维护性**：组件化开发，类型安全，代码注释清晰
