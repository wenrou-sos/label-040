# 充电桩实时状态查询与使用统计看板

## 技术栈

- **Vue 3** + **TypeScript**
- **ECharts**（图表可视化）
- **Naive UI**（组件库）

## 功能概览

### 充电桩分布地图
- 桩位标记点：绿色=空闲、红色=占用、灰色=离线
- 点击标记弹出信息卡（地址、电价、空闲枪数、当前充电功率）

### 列表模式
- 表格展示所有充电桩
- 支持按状态/运营商/距离筛选和排序

### 统计看板
- 今日充电量趋势折线图
- 各时段使用率热力图
- 各站点充电次数排行

### 个人中心
- 历史充电订单列表
- 充电量统计
- 费用统计

### 其他
- 适配手机端响应式布局
- Mock 数据模拟

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
